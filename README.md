# SalesWorkforce - (DevPost - QuickBase Hackathon 2020)

## Inspiration
I have a friend named John, he is a sales agent, he enjoys to go to his clientsâ€™ place to offer some products and go to the office to process them. But due to global pandemic, John having a hard time to do that., because of that, John loses some sales and become depress. With that, SalesWorkforce App was built, and made it possible using QuickBase and Azure. 

## What it does
- Allows Sales Agents to create a customer profile
- Allows Sales Agents to make a purchase order
- Sales Agents can receive status notification about their purchase order
- All data is stored on QuickBase.
